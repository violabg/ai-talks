# Commands Module

Load this module only when the user asks to run, debug, or validate commands.

- Package manager: `pnpm` only.
- Dev server: `pnpm dev`.
- Production build: `pnpm build`.
- Lint: `pnpm lint`.
- Tests: no test runner configured.
- Presentation narration draft: `npx tsx scripts/generate-speech-json.ts <slug>`.
